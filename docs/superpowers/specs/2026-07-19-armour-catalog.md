# Heretics 40K — Armour Catalogue

> First-pass, balance-tunable. Defense = per-element soak; a hit deals `max(0, damage − def[element])`.
> `0` in a column = that element bypasses this armour. Budget = sum of the six values.

**143 pieces** · 140 faction + 3 universal
